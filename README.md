# 🎓 BAC Prep AI - Platformă Pregătire Bacalaureat Matematică

Platformă web cu inteligență artificială pentru pregătirea Bacalaureatului la Matematică.

## 📋 Features

- 200+ exerciții reale de la BAC (2015-2024)
- AI personalizat - recomandări bazate pe ML
- Predicție notă BAC
- Adaptive difficulty
- Dashboard cu progres pe topicuri
- Simulări BAC complete

## 🛠️ Tech Stack

**Backend:**
- Python 3.14
- Flask
- PostgreSQL
- scikit-learn
- Surprise (recommender systems)

**Frontend:**
- React
- TailwindCSS
- Recharts

## 🚀 Setup

### Backend
```bash
cd backend
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
python app.py
```

### Frontend
```bash
cd frontend
npm install
npm run dev
```
